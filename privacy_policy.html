Privacy Policy

App Name: Prepaid Arrear

Developer: MD Jasim Uddin, Sub Assistant Engineer, BPDB

Last Updated: June 2026

1. Introduction

Prepaid Arrear is a utility app developed for looking up prepaid meter arrear information from the Bangladesh Power Development Board (BPDB) public billing system. This privacy policy explains how the app handles information.

2. Information We Collect

This app does not collect, store, or share any personal information. The only input required is a 12-digit prepaid meter number, which is used solely to query the BPDB public server for arrear information.

3. Data Storage

No user data is stored on any server or database by this app. No account registration is required. No login credentials are collected.

4. Internet Access

The app requires internet access to connect to the BPDB billing API to retrieve meter arrear information. This connection is made directly to BPDB servers only.

5. Third Party Services

This app does not use any third-party analytics, advertising, or tracking services.

6. Children's Privacy

This app is intended for use by BPDB staff and general electricity consumers. It does not knowingly collect information from children.

7. Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page.
